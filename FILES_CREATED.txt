GETTING STARTED
================================================================================

Step 1: Install dependencies
        python3 -m venv venv
        source venv/bin/activate (Mac/Linux)
        pip install -r requirements.txt

Step 2: Add your images
        - Profile photo to static/images/profile.jpg
        - Project images to static/images/projects/
        - Certificate images to static/images/certificates/

Step 3: Configure your data
        - Edit data/projects.json
        - Edit data/certificates.json
        - Update contact info in templates

Step 4: Add your CV
        - Place PDF in static/downloads/cv.pdf

Step 5: Run the app
        python3 app.py
        Visit: http://localhost:5000
